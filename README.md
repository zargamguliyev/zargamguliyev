## 3. [Автоматизация логистики прачечной (Oasis Retreats)](laundry-logistics-optimization)

Проект по бизнес-анализу, направленный на улучшение логистических процессов между отелями и централизованным прачечным комплексом.

Проект был обусловлен операционными сложностями, возникшими в результате быстрого расширения сети отелей, и направлен на автоматизацию планирования доставки постельного белья.

Основные активности:
- Анализ интервью с заинтересованными сторонами
- Определение системных требований
- Моделирование Use Case
- Диаграммы классов и объектов UML
- Подготовка технического задания

**Инструменты:** UML, Draw.io, Excel

[Смотреть проект →](laundry-logistics-optimization)
